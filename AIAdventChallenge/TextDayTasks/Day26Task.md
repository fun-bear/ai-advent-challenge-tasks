# Day 26 — Работа с Ollama (Windows + llama3.1:8b)

## Установка Ollama на Windows

1. Перейди на официальный сайт: [https://ollama.com/download](https://ollama.com/download)
2. Скачай установщик для **Windows**.
3. Запусти `.exe` и заверши установку (обычно достаточно стандартных настроек).
4. После установки открой **PowerShell** или **Command Prompt**.
5. Проверь, что Ollama установлен:

```bash
ollama --version
```

Если команда возвращает версию, значит всё готово.

## Загрузка модели llama3.1:8b

Скачай модель командой:

```bash
ollama pull llama3.1:8b
```

Ollama загрузит модель в локальное хранилище. Это может занять время в зависимости от скорости интернета.

## Включение (запуск) модели

Чтобы запустить модель в интерактивном режиме, выполни:

```bash
ollama run llama3.1:8b
```

После запуска откроется диалоговый режим в терминале.

## Пример отправки сообщения-диалога

После команды `ollama run llama3.1:8b` введи сообщение, например:

```text
Привет! Объясни простыми словами, что такое нейронная сеть.
```

Модель ответит прямо в терминале. Затем можно продолжать диалог следующими сообщениями.

---

### Быстрый сценарий целиком

```bash
ollama --version
```
