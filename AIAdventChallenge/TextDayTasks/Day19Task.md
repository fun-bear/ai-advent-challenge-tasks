# Day 19 — Отправка сообщения на Луну

## Конфигурация MCP Server для VS Code Cline

```json
{
  "mcpServers": {
    "ai-advent-mcp-server": {
      "url": "http://localhost:5279/",
      "type": "streamableHttp",
      "disabled": false,
      "autoApprove": []
    }
  }
}
```

## Исходный промпт для Cline

> Для отправки сообщения на Луну и получения ответа необходимо выполнить следующую цепочку действий: провести цензуру сообщения -> зашифровать сообщение -> отправить сообщение и получить ответ.

## Пример промпта

`Отправь сообщение на Луну: "Я люблю broccoli"`
